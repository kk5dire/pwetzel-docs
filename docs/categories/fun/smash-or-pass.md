# Smash-or-pass
Determines if a user is worthy of a smash or a pass

## Aliases
`pass-or-smash`, `smash-pass`, `pass-smash`
## Permit
General
### Bot Permits
`EMBED_LINKS`
### Cooldown
1 second
## Format
`smash-or-pass [user]`
## Syntax
`[user]` - Target of command (Optional)
## Examples
`,smash-or-pass @IYA`
